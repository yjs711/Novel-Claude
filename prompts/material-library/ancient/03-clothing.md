# 古代服饰 — 唐/宋/明 三朝服色等级

> 来源: 维基百科/北京法院网/搜狐历史
> 用途: 描写角色着装、社会阶层区分

## 核心原则
"上得兼下，下不得拟上" — 上级可穿下级服色，下级绝不可僭越。
《礼记》: "作异服以疑众，杀。"

## 唐朝官服 (品色衣制度)
| 品级 | 服色 | 腰带 |
|------|------|------|
| 三品以上 | 紫色 | 金玉带 |
| 四品 | 深绯(深红) | 金带 |
| 五品 | 浅绯(浅红) | 金带 |
| 六品 | 深绿 | 银带 |
| 七品 | 浅绿 | 银带 |
| 八品 | 深青 | 鍮石带 |
| 九品 | 浅青 | 鍮石带 |
| 庶人 | 黄色/白色 | 铜铁带 |

- 材质: 官员绫罗丝绸, 庶人麻布/粗葛布
- **高宗后禁赤黄色**(皇帝柘黄袍专用)
- 庶人外衣: 曳地不超2寸, 袖不超1尺3寸

## 宋朝官服
| 品级 | 服色 |
|------|------|
| 三品以上 | 紫色(佩鱼袋) |
| 五品以上 | 朱色(绯, 佩鱼袋) |
| 七品以上 | 绿色 |
| 九品以上 | 青色 |
- 庶人/商贾/伎术: 只许皂(黑)/白二色
- "借紫""借绯": 特许穿紫色/绯色(荣誉)

## 明朝官服 (最严厉)
- **补子制度**: 文官绣禽(仙鹤/锦鸡), 武官绣兽(狮/虎/豹)
- 禁用黄色: 柳黄/姜黄/明黄**全部禁**
- 庶人只用绢/素纱, 禁用锦绮/纻丝/绫罗
- 商贾止衣绢布, 农家一人为商→全家不得衣绸纱
- 僭用违禁品: 制造者与穿戴者**同罪**, 可能被视为谋反

## 写作要点
- 紫色=最高权力, 民间绝对不可穿
- 黄色=皇帝专色(唐高宗起)
- 平民=白色/皂色/麻布本色
- 一件衣服的颜色=行走的身份证
