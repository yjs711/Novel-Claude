# 古代服饰 — 唐/宋/明 三朝服色等级

> 来源: 维基百科/北京法院网/搜狐历史
> 用途: 角色着装/社会阶层

## 核心规则
"上得兼下, 下不得拟上" — 《礼记》"作异服以疑众, 杀"

## 唐官服(品色衣)
| 品级 | 服色 | 腰带 |
|------|------|------|
| 三品上 | 紫 | 金玉带 |
| 四品 | 深绯 | 金带 |
| 五品 | 浅绯 | 金带 |
| 六品 | 深绿 | 银带 |
| 七品 | 浅绿 | 银带 |
| 八品 | 深青 | 鍮石带 |
| 九品 | 浅青 | 鍮石带 |
| 庶人 | 黄/白 | 铜铁带 |
- 材质: 官绫罗/庶麻布。高宗后禁赤黄色
- 庶人衣: 曳地<2寸, 袖<1.3尺
- 男: 乌纱帽+胡服风。女: 双环望仙髻/惊鹄髻(盛唐高耸), 女扮男装多见
- 内衣: 诃子。不裹脚

## 宋官服
- 三品上紫/五品上朱(绯)/七品上绿/九品上青
- 庶人商贾: 只许皂(黑)白二色
- "借紫""借绯": 特许荣誉
- 女: 冠子花哨, 缠足兴起
- 北宋以紫为贵, 南宋重素装

## 明官服
- 补子: 文官绣禽(仙鹤/锦鸡/孔雀/云雁)/武官绣兽(狮/虎/豹)
- 禁用一切黄色(柳黄姜黄明黄全禁)
- 庶人: 绢/素纱, 禁锦绮/绫罗
- 商人: 止衣绢布(最严)
- 僭越者制造穿戴同罪, 可视为谋反
